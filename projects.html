Current Projects

Here's a look at some of the projects I'm currently developing.

LORA Communication Device

Developing a long-range, low-power communication device using LoRa technology. This project aims to create a resilient, off-grid communication network for local communities, ideal for emergency situations or areas with poor connectivity.

More Info

Disposable Vape Repurposing

An experiment in sustainability. This project involves salvaging screens and lithium-ion batteries from disposable vape pens to be repurposed in other low-power electronics, reducing e-waste and giving components a second life.

More Info

Indoor Food Cultivation

Researching and building compact, efficient indoor systems for growing produce, spices, and mushrooms. The goal is to source materials sustainably and develop methods for year-round urban farming, including microgreens.
